JS Loops

Loops can execute a block of code a number of times...

Loops has five types

  • For Loop
  • While Loop
  • Do while Loop
  • For...Of Loop
  • For...In Loop

Things need to be remembered

for loop == run known number of iterations
while loop == repeat until condition is false
do..while loop == like while but it runs at least once
for ... in loop == loops over object properties
for...of loop == loops over iterables (array , string , sets, map)
